readme: 4inaline.py

Description:
Program where user plays 4 in a line game against CPU that uses alpha-beta pruning to calculate best moves

Instructions:
1) To run this program, run the following on the command line: python 4inaline.py
2) Game will start up asking for setup details.
3) Enter answers in prompt of who will go first and time allotted to algorithm.
4) Game starts and then you play by entering board locations (i.e. b5)
5) Program will end when either player or CPU loses.
