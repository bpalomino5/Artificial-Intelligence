Readme: 8-puzzle.py

This program's main is divided into three parts.
1) Interactive Prompt
	This is on by default. Straight-forward use of program manually inputting 8-puzzles or selecting a randomnly generated one.

2) Table Data Generator
	Has comment blocks around it ("""). To use, uncomment section and change depth variable to desired value and i for number of puzzles wished to evaluate. 
	Program will then calculate average time using both heuristics and nodes generated along with 8 puzzles that have desired solution depth.

3) 3 Solution Path Calculator
	Has comment blocks around it ("""). To use, uncomment section. Will generate solution paths with each state in solution printed neatly to screen. 
	To modify, edit or add new puzzle to list of puzzles already provided.
