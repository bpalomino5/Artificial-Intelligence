Readme: nqueen.py

This program consists of one simple main program which solves the n-queen problem for any amount of tests.
To run this program edit the n-value to whatever size of board and queens desired and the size-value for the amount
of iterations to test with.The program will then solve i boards, calculating average elasped time and percentage of boards solved.

Save file and run in the terminal as: python nqueen.py
